# CRUD MEVN : MongoDB + Express + Vuejs + Nodejs

## Server

create config mongodb path

>> server/DB.js

```js
module.exports = {
    DB: 'mongodb://url'
}
```

start server

```sh
$ cd server 
$ npm install
$ npm run start
```

## Client

```sh
$ cd client 
$ npm install 
$ npm run serve
```

### Path View

clinet : localhost:8080
server api : localhost:4000/items
